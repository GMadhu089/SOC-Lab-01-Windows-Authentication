Executive Summary

A Windows authentication alert was triggered after multiple failed login attempts (Event ID 4625) were detected on a Windows 11 endpoint. The investigation was performed using Splunk Enterprise to determine whether the activity represented a brute-force attack, password spraying, or normal user behavior.
The investigation showed five consecutive failed login attempts followed by one successful login (Event ID 4624) from the same endpoint. No evidence of lateral movement, privilege escalation, or malicious PowerShell execution was observed. The activity was determined to be a user entering an incorrect password multiple times before successfully authenticating.
